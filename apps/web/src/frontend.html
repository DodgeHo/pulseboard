FAQ

Questions a buyer would actually ask.

A good portfolio site should answer business questions first, then make technical proof easy to verify.

Is this only a visual prototype?

No. The commercial site is static, but it links to a live backend proof surface with health/readiness probes, OpenAPI, docs, and protected API behavior.

Why use a 3D building for a backend demo?

The building turns abstract platform layers into something memorable: exterior equals customer experience, floors equal backend modules, and the base equals infrastructure.

Can this be deployed to staging?

Yes. The repository includes Docker Compose deployment materials and public surface verification scripts for safe staging rehearsal.
